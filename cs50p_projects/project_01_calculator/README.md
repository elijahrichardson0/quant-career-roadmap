# Python Calculator

## Overview
This project is a command-line calculator built in Python as part of my CS50P practice projects.

The calculator allows users to perform basic arithmetic operations while handling invalid input gracefully.
## Features
- Addition
- Subtraction
- Multiplication
- Division
- Decimal number support
- Input validation
- Division-by-zero protection
- Ability to perform multiple calculations in the same session
- Support for uppercase and lowercase menu inputs

## How to Run
1. Open a terminal in the project directory.
2. Run:

python main.py

3. Follow the prompts in the terminal.
## Concepts practiced
- Variables
- User input
- Loops
- Conditionals
- Exception handling
- Program flow control
- Basic arithmetic operations
## Example Usage
Enter the first number: 10

Enter the operator: *

Enter the second number: 5

The answer is 50.0
