# Number Guessing Game

## Overview
This project is a command-line number guessing game built in Python as part of my CS50P practice projects.

The program randomly selects a number between 1 and 100, and the player must guess the correct number. After each guess, the program provides feedback indicating whether the guess was too high or too low.
## Features
- Random number generation
- Input validation
- Range validation (1–100)
- High/low feedback after each guess
- Guess counter
- Win condition detection
- Option to play multiple games
## How to Run
1. Open a terminal in the project directory.
2. Run:

python main.py

3. Follow the prompts to play the game.
